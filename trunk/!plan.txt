1. Минимизация загрузки по JSON.
   Загрузка только координак точек, подгрузка дополнительной информации по запросу (выбору точки).
2. Оптимизация трека при изменении масштаба.
3. Поддержка треков любой длины.
4. Geo-информация по точке (адрес).
5. Информация о времени остановки/стоянки. Объединение точек стоянки.
6. Информация о пройденом пути по отображенному треку.
7. Отображение по дням, по сменам (например с 8:00 до 8:00).
8. Упаковка старых треков.


Посмотреть идеи в http://code.google.com/p/geodatastore/wiki/HowItWorks и в исходниках
